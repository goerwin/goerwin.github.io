“Tiendecita.co” was a freelance project I worked on for local entrepreneurs in Colombia. The goal was to create a website for multiple restaurants with a custom, easy-to-use CMS.

I handled the architecture and setup, using HTML, CSS, JavaScript, and a basic CMS that generated page content from Markdown files.

My responsibilities were:

- Create the architecture and managing of pages using Grunt.js
- Create landing, product, history and contact pages
- Work on the CMS application using static content in markdown
